HTML - язык разметки.

Теги бывают:

  1. Одиночные;
  2. Парные:
    • У парных тегов есть открытый и закрытый тег;
    • Внутри парных тегов можно добавлять ещё теги.

Одиночные теги:

            <mark src="" width="" height="">
            <input id="" class="" name="">

Парные теги:

            <div> какой-то текст </div>
            <p> какой-то <i>очень интересный</i> текст </p>

Правильный пример вложенности:

            <div>
              <p>
                Какой-то текст
              </p>
            </div>

Ошибочный пример вложенности:

            <div>
              <p>
                Какой-то текст
              </div>
            </p>

Атрибуты:

  1. Все атрибуты пишутся внутри открытого тега.

Добавление атрибута:

            <тег атрибут=”значение” атрибут2=”значение” атрибут3=”значение”>
            <img src=”#” width=”10” height=”10” class=”class-01”>
             
            <тег атрибут=”значение”> </тег>
            <div class=”class-02”> </тег>

Где проверить на вложенность?

  1. html-tree - проверка на вложенность

Где увидеть список тегов?

  1. html-tags - список html тегов
  2. Но чаще всего встречаются: div, p, section, h1-h6, style, strong, b, i и т.д.

CSS - язык описания
внешнего вида.

Стандартная конструкция описания стилей:

            селектор {
              свойство: значение;
              свойство: значение;
              свойство: значение;
              свойство: значение;
              и т.д.
            }

Селекторами могут являться:

  1. tag (тег)
  2. class (класс)
  3. id (идентификатор)
  4. даже можно по атрибутам искать

Селектор - тег

            тег {
              свойство: значение;
            }
             
            div {
              background: #ffffff;
            }

Селектор - атрибут

            [атрибут="значение"] {
              свойство: значение;
            }
             
            [width="10"] {
              background: #ffffff;
            }

Селектор - класс

            .название-класса {
              свойство: значение;
            }
             
            .classname-01 {
              background: #ffffff;
            }

Селектор - идентификатор

            #идентификатор {
              свойство: значение;
            }
             
            #id-01 {
              background: #ffffff;
            }

Селектор - микс, тег + атрибут

            тег[атрибут="значение"] {
              свойство: значение;
            }
             
            div[data-attr="first"] {
              background: #ffffff;
            }

Селектор - микс, класс + атрибут

            .название-класса[атрибут="значение"] {
              свойство: значение;
            }
             
            .classname-01[data-attr="first"] {
              background: #ffffff;
            }

Селектор - микс, идентификатор + атрибут

            #идентификатор[атрибут="значение"] {
              свойство: значение;
            }
             
            #id-01[data-attr="first"] {
              background: #ffffff;
            }

Селектор - микс, тег + класс

            тег.название-класса {
              свойство: значение;
            }
             
            div.classname-01 {
              background: #ffffff;
            }

Если свойства CSS одинаковые, то селекторы можно перечислять через запятую, а именно:

            селектор,
            селектор,
            селектор {
              свойство: значение;
            }

Живой пример:

            .f-subheader,
            .f-header,
            .f-description {
              font-family: ‘Roboto’;
              font-size: 16px;
              color: #000;
            }

Это тоже самое, что и:

            .f-subheader {
              font-family: ‘Roboto’;
              font-size: 16px;
              color: #000;
            }
             
            .f-header {
              font-family: ‘Roboto’;
              font-size: 16px;
              color: #000;
            }
             
            .f-description {
              font-family: ‘Roboto’;
              font-size: 16px;
              color: #000;
            }

Почему в цвете я указал значение из 3 цифр, а не из 6?

  1. Давайте посмотрим на простой пример, возьмём цвет #ff00ff;
  2. Теперь разделим на 3 части;
  3. Получим ff / 00 / ff ;
  4. В каждой части, после первого символа, следующий повторяется;
  5. И тогда можно сократить, где получим f0f и сам цвет #f0f

Классы в тегах можно миксовать, такое часто можно увидеть на Геткурс:

            Один класс говорит о том, что это какой-то заголовок.
            А второй о том, что размер будет 52
             
            <div class="f-header  f-header-52"> какой-то текст </div>

Зачем 2 класса?

  1. Первый - основной класс;
  2. Второй - модификатор;
    • Если вы установите в конструкторе другой размер у заголовка, то
      f-header-52 пропадёт;
    • И появится новый;
    • А первый класс останется прежним.

Селектор - микс, несколько классов

            .название-класса-1.название-класса-рядом-1 {
              свойство: значение;
            }
             
            .f-header.f-header-36 {
              background: #ffffff;
            }

Но если:

  1. Один из этих классов не будет у элемента, то стили не применятся;
  2. Поэтому используйте основной.

Представим!

  1. Что, если если у нас будет много заголовков;
  2. И мы по классу .f-header заменим цвет на красный!

!IMPORTANT

            !important - это способ усилить специфичность вашего селектора
            специфичность мы разбирать не будем, про это будет статья

            Лучше всего разобраться в специфичности селекторов.

Как же всё-такие обратиться к нужному заголовку?

  1. Добавить стили внутрь блока;
    • Если заголовков в данном блоке будет несколько, то стили применятся ко всем в этом блоке
  2. Добавить уникальный класс
    • Тогда нам можно будет не беспокоиться о том, что какой-то заголовок перекрасится;
    • Покрасится только тот, который нужен.

Давайте рассмотрим на примере

CSS Selectors

  1. Селекторы - очень тонкая штука, про них действительно можно очень много сказать.
    • Например, что у них есть разные состояния
    • Псевдоэлементы
    • Псевдоклассы - про это мы сегодня поговорим

Представим ситуацию на Геткурс!

  1. У нас есть блок 3 колонки
  2. А стилизовать нужно лишь вторую
  3. Тут и можно использовать один из таких псевдоклассов :nth-child(n).

Как это может выглядеть?

  1. Найдём путь до 3х колонок, с указанием их общего контейнера
  2. Добавим псевдокласс nth-child(n), где n - это номер колонки
  3. И стилизуем её!

Но для начала посмотрим на конструкцию в целом.

ПСЕВДОКЛАССЫ!

            .уникальный-класс .родитель .колонки:псевдокласс(номер колонки) {
              свойство: "значение";
            }

PSEUDOCLASSES!

            .why-course-01 .row .lt-col:nth-child(2) {
              background: red;
            }

Что случилось? Почему цвет не поменялся?

  1. Дело в том, что когда мы добавлям наш код в CSS самого блока. То стили применяется уже к уникальному ID, который добавляет Геткурс.

А зачем мы добавляли класс?

  1. Чтобы задать уникальность блоку, например, если вы все стили добавите вне этого блока, как на примере дальше.

Какая гадость эта ваша заливная рыба эти ваши селекторы!

  1. Ничего страшного, с практикой вы запомните многие вещи и сможете сами их применять :)

Какие состояния и ещё псевдоклассы существуют?

  1. Их очень много, про все я говорить не буду, но разберём основные;
  2. Состояния:
    • :hover - наведение по элементу
    • :active - нажатие
    • :focus - фокус по элементу
    • :disabled - заблокированное состояние (например, если кнопка не является активной)
  3. Больше псевдоклассов!
    • :first-child - первый элемент
    • :last-child - последний элемент
    • :nth-child(n) - выбранный вами элемент, где n - его номер

И это ещё не весь список!
